# NC1_FormatDescription
## 소개
이미지와 비디오 포맷은 상당히 다양하고 각각의 포맷은 쓰임새가 있습니다.

포맷의 종류는 뭐가 있는지, 특징은 뭔지 잘 모르는 분들을 위해 간단한 설명을 해주는 앱입니다.
